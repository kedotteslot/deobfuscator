À propos du Désobfuscateur

Cet outil permet de désobfusquer du code JavaScript en utilisant diverses techniques d'analyse et de transformation.

Il peut détecter et traiter plusieurs types d'obfuscation :

  • Encodage hexadécimal (\x41)
  • Encodage Unicode (\u0041)
  • Chaînes encodées en Base64
  • Utilisation d'eval() ou new Function()
  • Renommage de variables
  • Compression de code
